# dotfiles

Personal dotfiles for macOS.

## Contents

- `zsh/.zshrc` - Zsh config with aliases for eza, bat, fzf, zoxide, lazygit, zellij
- `zellij/config.kdl` - Zellij config with vim-style keybindings
- `zellij/layouts/` - Project-specific layouts
- `CHEATSHEET.md` - Quick reference for CLI tools

## Install

```bash
./install.sh
```

Symlinks dotfiles to their proper locations.

## Dependencies

Install via Homebrew:

```bash
brew install eza bat fzf zoxide lazygit zellij
```
